# Recommend System

### Architecture

The front-end project is located in `front` folder.

`SeleniumIDE` generated python scripts can be found in `uiTest` folder.

### Setup

```
cd front
```

```
npm install
```

```
npm run serve
```

#### run Jest testing code

```
cd front
```

```
npm test
```

#### run python scripts

~~~
cd uiTest
~~~

~~~
python AnyOfThePythonFile.py
~~~

Note that you should make `addFavourite.py` run before `deleteFavourite.py`.
